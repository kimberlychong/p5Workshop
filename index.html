The Basics

Programming/Coding: Act of writing instructions for the computer to perform various tasks.

Programming Language: Set of commands, rules, or syntax for instructing a computer to perform specific tasks.

Syntax: Rules that specify the correct combined sequence of symbols that form a structured proram using a programming language.

Today we'll learn how to instruct the computer to create images!

What is JavaScript?

JavaScript is among the most powerful and flexible programming languages of the web. It is also one of the core technologies of the web, along with HTML and CSS. It powers the dynamic behavior on most websites.

What is p5.js?

p5.js is a JavaScript library, collection of predefined commands and logic, with a full set of drawing functionality.
